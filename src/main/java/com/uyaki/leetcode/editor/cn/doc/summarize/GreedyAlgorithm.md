# 贪心算法

## 本质

局部最优解 -> 全局最优解

## 基本步骤

- 步骤1：从某个初始解出发； 
- 步骤2：采用迭代的过程，当可以向目标前进一步时，就根据局部最优策略，得到一部分解，缩小问题规模； 
- 步骤3：将所有解综合起来。
